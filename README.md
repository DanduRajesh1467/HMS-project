# 🏨 Hotel Management System

## 📌 Overview

This is a web-based Hotel Management System developed using Python and Flask. It helps manage room bookings, customer details, and basic hotel operations.

## 🚀 Features

* Add new bookings
* View all bookings
* Delete bookings (checkout)
* Simple and user-friendly interface
* Data stored using JSON file

## 🛠️ Technologies Used

* Python
* Flask
* HTML
* CSS

```
├── static/
│     └── styles.css
```

---

## ▶️ How to Run

1. Install Flask:

```
pip install flask
```

2. Run the application:

```
python app.py
```

3. Open browser and go to:

```
http://127.0.0.1:5000/
```
